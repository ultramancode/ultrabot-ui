# 울트라챗봇 (UltraChatbot)

울트라챗봇은 사용자 경험에 최적화된 AI 챗봇 UI 프로젝트입니다.  
LLM 기반 대화 흐름에 특화되어 있으며, API 호출 전 버튼 기반 확인, 로딩 메시지 처리 등 다양한 기능을 제공합니다.

---

## 🌟 주요 기능

- ✅ 예/아니오 버튼 기반 API 호출 확인 흐름
- ✅ LLM 백엔드 직접 연동 (로컬 LangChain 등과 연결 가능)
- ✅ 깔끔한 컴포넌트 기반 구조

---

## ⚙️ 실행 방법

```bash
pnpm install
pnpm dev
```

## 🛠 개발 스택
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
